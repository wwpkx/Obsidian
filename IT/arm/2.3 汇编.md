# 概述
## 为什么使用汇编
- 初始化
- 效率

## 分类
- ARM 标准汇编
	- 适用于ARM公司的汇编器
	- 适合在Windows平台下使用，如ADS中使用
- GNU 汇编
	- 适用于GNU交叉编译工具链中的汇编器
	- 适合于Linux开发平台

## 汇编程序框架
```
.section .data
	< 初始化的数据>
.section .bss
	< 未初始化的数据>
.section .text
.global _start
_start:
	<汇编代码>
```

```
.text
.global _start
_start:
	<汇编代码>
```

# 汇编指令
## 算术和缓辑指令
- mov
- mvn
- sub
	- SUB     R0, R1, R2              ; R0 = R1 - R2
- add
	- ADD     R0, R1, R2              ; R0 = R1 + R2
- and
	- AND     R0, R0, #3              ; R0 = R0 AND 0b11
- bic 位清除
	- BIC     R0, R0, #%1011          ; 清除 R0 中的位 0、1、和 3。保持其余的不变。

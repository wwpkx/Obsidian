# 概述
## 为什么使用汇编
- 初始化
- 效率

## 分类
- ARM 标准汇编
	- 适用于ARM公司的汇编器
	- 适合在Windows平台下使用，如ADS中使用
- GNU 汇编
	- 适用于GNU交叉编译工具链中的汇编器
	- 适合于Linux开发平台

## 汇编程序框架
```
.section .data
	< 初始化的数据>
.section .bss
	< 未初始化的数据>
.section .text
.global _start
_start:
	<汇编代码>
```

```
.text
.global _start
_start:
	<汇编代码>
```

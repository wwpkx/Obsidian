# 总流程
- 在主机上编译Bootloader, 然后通过JTAG烧入
	- **将BootLoader烧入到 nor flash中**（驱动、内核等都在**nand flash**中）
	- 然后再通过BootLoader操作其他烧入，比如内核，驱动
	- 从**nand flash启动**，测试驱动等
- 在主机上编译嵌入式Linux内核，通过Bootloader烧入单板
	- linux内核支持nfs
	- 然后通过nfs调试各种驱动或者软件
	- 通过后，再烧入

# 内核编译
![](../photo/Pasted%20image%2020230502152312.png)
```
1、清除原有配置与中间文件
x86: make distclean
arm: make distclean

2、配置内核
x86: make menuconfig 
arm: make menuconfig ARCH=arm

3、编译内核
x86: make bzImage
arm: make uImage ARCH=arm CROSS_COMPILE=arm-linux-
```

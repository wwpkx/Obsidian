# 说明
- 不同IDE集成的**make遵循的规范和标准不同**，导致其语法、格式不同

![](../photo/Pasted%20image%2020230228094352.png)

# 使用
- 写CMakeLists.txt
- 使用命令 **cmake .**  //目的，生成makefile文件
	 - 内部构建
		- 编译文件与源文件混杂在一起
	- **外部构建（推荐）**
		- 新建build文件夹，在build文件夹中执行 **cmake ..**
- 使用命令 **make**
- 使用命令 **make install**

# 语法
- 变量使用 ${} 取值
- 指令(参数 1 参数 2...)
- 指令不区分大小写（推荐大写），参数和变量区分大小写

# 扩展名

- .c    需要预处理的 C 语言源代码
- .h    C 语言源代码头文件
- .i    不需要预处理的 C 语言源代码
- .s    汇编语言代码
- .o    目标文件
- .a    静态对象库
- .so   共享对象库

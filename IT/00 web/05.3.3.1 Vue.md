# Vue.js 是什么
- 是一套构建用户界面的**渐进式框架**
- Vue 采用**自底向上增量开发的设计**
- Vue 的核心库**只关注视图层**
- Vue 完全有能力驱动**采用单文件组件**和**Vue生态系统支持的库**开发的**复杂单页应用**
- Vue.js 的目标是通过尽可能简单的 API 实现**响应的数据绑定**和**组合的视图组件**

# Vue.js 安装
- `$ npm install vue`

# Vue CLI 
- Vue CLI 致力于**将Vue生态中的工具基础标准化**
- 搭建**项目脚手架**
- 快速开始**零配置原型开发**

# Vue CLI 安装
- `npm install -g @vue/cli`

# 挂载
- mount /mnt/cdrom，挂载光驱
- unmount /mnt/cdrom，卸载光驱

# 分区
- 基本分区和扩展分区的数目之和**不能大于4个**
- **基本分区（Primary Portion）**，**不能再分区**
- **扩展分区（Extension Portion）**，**进行分区后才能使用**
	- **逻辑分区（Logical Portion）**，而且逻辑分区没有数量上限制

# df（disk free）
- 显示磁盘分区上可以使用的磁盘空间
	- -a    // 查看全部文件系统，单位默认KB
	- -h    // 使用-h选项以KB、MB、GB的单位来显示，可读性高

# du (disk usage)
- 显示每个文件和目录的大小
	- -h  // 以K  M  G为单位显示，提高可读性
	- --max-depth=1  // 显示层级

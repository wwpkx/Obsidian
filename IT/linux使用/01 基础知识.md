# 关机命令
```
reboot 现在重新启动计算机

shutdown -h now 立即进行关机
shutdown -r now 现在重新启动计算机
	-t sec : -t后面加秒数,即”过几秒后关机”
	-k	 : 不是要真的关机,只是发送警告信息
	-c	 : 取消已经在进行的shutdown命令内容
	-n	 : 不经过init程序,直接以shutdown关机
	-f	 : 关机并启动之后,强制略过fsck的磁盘检查
	-F	 : 系统重新启动之后,强制进行fsck的磁盘检查
	-c	 : 取消已经在进行的shutdown命令内容
```

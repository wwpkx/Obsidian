# 分工模式
- 架构师做EIT设计
	- E是控制点，透过I来驱动T
	- **E+I = 框架(Framework)**
	- **T = 插件(Plug-in)**
- 强龙做E&I，做框架
	- 强龙掌控系统面：E是控制点，透过I来驱动T
	- 强龙掌控生产面：强龙掌控I，外包就不会失控
- 地头蛇配T，配插件

- -Oriented
	- 意味着一种**信仰**
	- Object-oriented， **相信任何**软件都是由**对象**所构成的，而且**Nothing else**
	- 面向过程（procedure-oriented）
- -Based 
	- 有先后的顺序
	- Requirement-based，**基于需求**而进行后续的软件开发活动
- -Driven
	- **引导，驱动**
	- Model-driven
	- Use Case-driven
- -Centered
	- Architecture-centered，一切软件开发的活动都**围绕着**架构

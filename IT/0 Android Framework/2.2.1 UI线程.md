- UI线程的责任：**迅速处理UI事件**
- 超过5秒钟而未处理的话，弹出警告框
- 默认情形下，Activity、BroadcastReceiver或Service由**主线程**执行
	- AMS执行，分析AndroidManifest.xml
	- 向Message Queue中发送 create 信息
	- Main Loop执行 Activity.onCreate

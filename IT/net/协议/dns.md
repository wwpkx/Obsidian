# 概念
-   Domain Name system  域名解析系统 （域名为了方便人们记忆
-   将域名解析为IP地址
-   分为 **请求报文request** 和 **响应报文relay**
-   好用的DNS服务器：**114.114.114.114, 223.5.5.5**

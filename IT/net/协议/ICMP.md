# 概念
- Internet control massage protocol 因特网**控制**协议
- 在主机与路由器之间传递**控制信息**，包括报告错误、交换受限控制和状态信息等
- 基于 ICMP 实现的命令
	- ping 测试网络**可达性，延迟，丢包**
	- tracert 显示经过的**路径**
